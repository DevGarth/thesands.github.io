What's an NFT

A NFT (Non-fungible token) is a file with a unique signature on the Blockchain. It can be an image file, a song, a tweet, a text posted on a website, a physical item and various other digital formats.
NFT’s provides a path to have clear ownership over digital items. You can buy, sell, trade or give NFT’s to anyone. Value of the NFT might change overtime, It's based on what people are willing to pay for it.

How's NFT stored?

Each NFT is an ERC-721 Ethereum Token stored on the Ethereum blockchain. The Metadata associated with each NFT is stored on IPFS.
This is a decentralized server which ensures that the content will never change and will always be available, whilst also reducing on-chain data and gas fees.
